CSS

웹 페이지를 아름답게 만드는 코드.
웹을 꾸미기 위해 html에 디자인에 대한 새로운 태그를 추가하는 방법을 고안했다.
이전에 쓰던 방식은 <.font>라는 태그를 사용했다.

글꼴 색 변경을 원하는 글자의 앞 뒤에 태그를 사용한다. (ex. <.font color="색상">글자<./font>)

그러나 이 태그에는 문제가 있다.
첫째, 시각장애s인이 보았을 때 의미가 없는 것들은 정보가 아니게 된다.
둘째, font 태그는 각 글자마다 적용을 시켜주어야 하는데 이것이 계속 중복되면 매우 복잡해질 것이다.

그러므로 어렵고도 실효성 있는 CSS를 배워보도록 한다.



기본 설정

이 코드가 CSS임을 표기

웹브라우저는 기본적으로 코드를 html이라고 생각한다. (가장 기본적인 코드였기 때문.) 그러므로 head에 html의 문법으로 이 코드가 css임을 알려주어야 한다. 태그 형식은 다음과 같다.


        <태그>
          a {
            내용(property:value)
          }
        < /태그>

기존 html 태그에 속성을 부여하는 경우도 있다. 그런 경우 형식은 다음과 같다.

< html태그 코드="속성(property:value)">

컴퓨터가 무시하도록 명령

걍 혹시 몰라 남겨둠

코드 파일에만 존재하고 웹에서는 구현을 시키고 싶지 않은 경우가 있다. 이런 경우 사용하는 태그는 다음과 같다. 


            < !-- (내용) -->

부가설명

head에서 style 코드 내에 부가설명을 남기고 싶을 때 사용하는 태그는 다음과 같다.


            < style>
              /*
              내용
              */
            < /style>

style 태그 연결하기

하나의 웹페이지에 연결되는 html의 css 코드를 통일시키기에 아주 좋은 태그로 link 태그가 있다.


          < link rel="stylesheet" href="(css파일)">

위와 같이 사용하면 되는데, 원래 head에 style 태그로 적어야 했던 것을 css파일에 style 태그만 옮겨두면 된다.
이렇게 처리해두면 페이지의 공통된 레이아웃을 수정할 때 따로 빼둔 css파일의 style 태그만 수정하면 되기 때문에 경제성이 매우 높아진다.

선택자(Selector)

선택자는 '그룹' 개념과 유사하다.
차이가 있다면 하나의 요소를 여러 그룹에 소속시킬 수 있다는 점이다. 어떻게 여러 그룹에 소속시키고 해당 그룹에 어떻게 속성을 부여할 수 있는지 알아보자.

선택자는 기본적으로 head와 연동시키는 방식으로 작동한다.
head에 작성된 코드는 일종의 control tower이고, 각 요소에 포함된 a 속성값들은 각각 해당 control tower에 일원이라는 증표와 같다.

class 태그

가장 기본적인 선택자 태그다.
그룹에 소속시키고 싶은 단어나 문장에 a 태그를 작성하고 class="(그룹명)"을 적어서 소속을 시킨 다음
head로 올라가서 style 태그 안에 .(그룹명) 이후 코드를 작성해주면 된다.
예시는 다음과 같다.


          < style>
            .group{
              color:gray;
            }
            .team{
              color:red;
            }
          < /style>

          < a class="group ">문장< /a>
          < a class="group team">단어< /a>

class 태그는 같은 단어/문장에 여러 개를 등록시키는 것도 가능한데, 코드가 중복되는 경우엔 head에서 단어/문장과 가까운 순서로 우선순위가 결정된다. 따라서 위 코드에서 단어는 red로 표시된다.

id 태그

id 태그는 class 태그에 좀 더 위계를 두는 태그이다. 다만 이런 경우엔 팀 추가시 .이 아닌 #을 추가하고, 태그에 적을 때엔 class가 아닌 id를 입력한다.
이것을 작성해보면 다음과 같다.


        < style>
          #group1{
            color:gray;
          }
          .group2{
            color:red;
          }
        < /style>

        < a class="group2" id="group1">단어< /a>

위와 같은 경우에는 group1이 더 우선순위를 갖게 되므로 group1과 2 모두 속해있는 단어는 gray로 표시될 것이다.

글자 설정

글꼴 색상 변경

글꼴 색상 변경하는 방법에는 두 가지가 있다.

  1. head에 <.style> 태그 사용하기
  2. a에 style 속성 추가하기

1. style 태그 사용하기

head에 태그를 추가하는 방식으로 사용한다. 형식은 다음과 같다.

<.style>
a{
color:(색상);
}
<./style>

2. style 속성 추가하기

보통 a 태그 안에 속성을 부여한다. 방식은 다음과 같다.

<.a style="color:(색상);">글자<./a>

텍스트 꾸밈 없애기

글꼴 꾸밈을 없는 방법에는 두 가지가 있다.

  1. head에 <.style> 태그 사용하기
  2. a에 style 속성 추가하기

1. style 태그 사용하기

head에 태그를 추가하는 방식으로 사용한다. 형식은 다음과 같다.

<.style>
a{
text-decoration:none;
}
<./style>

2. style 속성 추가하기
*이미 속성이 있을 경우엔 띄어쓰지 않는다. (ex. <.a style="color:blue;text-decoration:underline")

보통 a 태그 안에 속성을 부여한다. 방식은 다음과 같다.

<.a style="text-decoration:(속성) 혹은 none;">글자<./a>

이미 해당 링크에 접속했을 때 표시되는 글자 색 변경하기

보통 전체적으로 적용시킬 때에는 html 태그 중 하나인 class 태그를 사용한다.

1. 링크가 첨부된 단어의 a 태그에 class="그룹이름" 작성
ex. <.a href="1.html" class="saw">HTML<./a>
2. head의 style 태그에 가서 .그룹이름 후 속성 부여.
ex. <.style>
a{
.saw {
color:(색상);
}
<./style>

* 하나의 단어에 그룹을 여러가지 추가하고 싶을 경우 <.a class="(그룹1) (그룹2)"> 형식으로 입력하면 된다.

레이아웃 설정

우리는 내용의 위계를 구분하기 위해서 p 태그와 h 태그로 글의 위계를 두곤 하지만, 보통 웹은 라인으로 레이아웃을 구성하곤 한다. 어떻게 레이아웃을 구성하면 좋을지 박스타입부터 알아보자.

박스 타입 설정

박스 타입을 설정하기 전에, 각 글자/문장/문단이 웹 내에서 얼마만큼의 부피를 차지하고 있는지 이해를 한 상태로 코드를 짜야한다.
코드는 head에 작성해야 하고, 형식은 다음과 같다.


        < style>
          /*
          block level element
          */
          h 태그{
            border-width:(수)px;
            border-color:(색상);
            border-style: (성격);
            padding:(수)px;
            margin:(수)px;
            width:(수)px;
          }
          /*
          inline element
          */
          a 태그{
            border-width:(수)px;
            border-color:(색상);
            border-style: (성격);
          }
        < /style>

박스 라인 설정

박스 라인을 일부분만 넣고 싶은 경우 


            태그 {
              border-bottom:(수)px solid (색상);
            }
다른 방향에 라인을 넣고 싶은 경우 방향을 바꾸면 된다. 

            border-top
            border-right
            border-left
            border-bottom

display 설정

h 태그의 경우엔 block(웹의 가로축을 꽉 채워서 존재), a 태그의 경우는 inline(딱 부분만 존재) 형식으로 존재하지만 이런 기본적인 성질을 변화시키고 싶은 경우엔 다음과 같이 display 설정을 넣으면 된다.


          < style>
          h 태그{
            display:inline;
          }

위와 같이 작성하면 h 태그 바로 옆에도 글을 작성할 수 있다.
이외에 부여 가능한 성질은 다음과 같다.

  • display:inline; : 그 내용에 해당되는 만큼의 박스 형성
  • display:block; : 그 내용이 존재하는 웹의 가로축 전체를 꽉 채워서 형성
  • display:none; : 아예 존재를 없앤다.

태그 생략

위의 태그는 너무 길어지니까 이를 축약하기 위해서 태그에 ,(쉼표)를 붙여서 사용하기도 한다. 


        < style>
          h1, a{
            border:5px solid red;
          }
        < /style>

개발자 도구 : 검사

해당 페이지가 어느정도의 margin, padding, width 등을 가지고 있는지 보기 위해서는
화면을 우클릭한 뒤 '검사'를 클릭해서 확인하면 된다.

미디어 쿼리

미디어 쿼리란 화면 비율이 바뀌었을 때 비율에 따라 형태가 바뀌는 유동적인 레이아웃 형태를 말한다. 


          < style>
            @media(max-width:(수)px) {
              (변경하고 싶은 내용)
            }
          < /style>

max-width 뒤에 들어가는 숫자는 페이지의 가로폭이 숫자 이상으로 넘어가면 형태가 바뀔 것이라는 의미다.

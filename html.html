HTML



태그

Html의 명령어 같은 존재. '<>'로 명령을 시작하고 ''로 끝내는 것이 일반적이다.

html 구조 구분하기

html을 작성하기 전, 전체적인 구조를 구분할 때에 html, head, body라는 총 3가지의 태그를 사용한다. 가장 상위개념은 html이고, title이나 폰트를 사용하는 charac 태그를 포함하는 것이 head, 그 이하 본문에 해당하는 내용이 모두 body의 이하에 들어간다.

제목 표시하기

제목을 쓸 때에는 title 태그와 h(숫자) 태그를 사용해서 구분한다. 전자는 웹페이지 자체의 제목을 설정하고, 후자는 웹페이지 내에서 사용할 대제목, 중제목, 소제목으로 사용할 태그이다. 숫자가 커질수록 소제목이 된다.
(ex. <.h1>제목<./h1>)

줄바꿈

줄바꿈을 사용할 때에는 br이나 p 태그를 사용하는데, 전자는 단순한 줄바꿈이고 후자는 phragraph의 준말이므로 단락을 띄울 때 쓰는 단위이다.
줄을 띄어쓰는 정도에 대해서, br 태그를 사용할 경우엔 태그를 반복해서 사용하면 되고 (ex. <.br><.br><.br>)p 태그를 사용할 때에는 style="margin-top:(단위)px;"을 사용하여 조절하면 된다. 단, 이 태크는 시작태그에 붙여야 한다.
(ex. <.p style="margin-top:45px;")

이미지 삽입

이미지를 삽입할 때 쓰는 태그는 img 태그이다.
img src="(이미지 링크)" 형식으로 사용하면 되고, 크기 조절은 width="(크기 혹은 퍼센트)"을 이어붙이면 된다.
(ex. <.img src="coding.jpg" width="100%")

목록 정리하기

목록을 단순히 사용하는 경우에는 ul태그와 li태그를 같이 사용한다.
ul태그가 부모 태그, li 태그가 자식 태그가 되어서 실제 사용하는 경우엔
<.ul>
<.li>내용<./li>
<.li>내용<./li>
<./ul>
형식으로 사용한다.
결과는 이렇게 나온다.

  • 내용
  • 내용

만약 숫자로 정리하고 싶을 경우에는 ol태그와 li태그를 같이 사용해주면 되는데 이땐 ol태그를 부모 태그로 사용해주면 된다.
예시는 다음과 같다.
<.ol>
<.li>내용<.li>
<.li>내용<./li>
<./ol>
결과는 다음과 같다.

  1. 내용
  2. 내용

표 만들기

표는 영어로 table. 따라서 table 태그와 tr,td 태그를 사용한다.
table태그를 가장 큰 묶음으로 사용하고 그 밑으로 tr, td 태그를 사용하면 되는데, 예시는 다음과 같다.
<.table>
<.tr>
<.td>목차1<./td>
<.td>내용<./td>
<./tr>
<.tr>
<.td>목차2<./td>
<.td>내용<./td>
<./tr>
<./table>
이를 실제로 적용하면 결과는 다음과 같다.

목차1 내용
목차2 내용

여기서 표의 디자인을 가미하기 위해서는 다양한 명령어가 추가로 필요하다.

  1. border 명령어는 표의 테두리를 설정한다. border="(값)"에서 값이 올라갈수록 테두리가 두꺼워진다. 최소값은 1이다.
    표 테두리를 한 줄만 설정하려면 CSS 내용 중에 있는 border-collapse 값을 collapse로 지정하면 된다. 예시는 border-collapse:collapse; 형식이다. 나누려면 separate라고 하면 된다.
  2. bordercolor 명령어는 표 테두리의 색상을 지정한다. bordercolor="(색상)" 형식으로 사용하고, 기본값은 검정이다.
  3. width와 height 명령어는 테이블의 크기을 결정한다. width=50px width=100% 형식을 사용해 크기, 비율을 수정할 수 있다.
  4. align 명령어는 테이블 안에 있는 값들을 정렬시킨다. align="center" 형식으로 적용한다.
  5. bgcolor 명령어는 원하는 태그의 배경색을 변경시킬 수 있다. bgcolor="(색상)" 형식으로 사용한다.
  6. colspan과 rowspan은 셀을 합병시키는 기능을 한다.

링크 연결하기

링크를 연결할 때에는 갈고리를 의미하는 anchor의 첫 문자, a 태그를 사용한다.
예를 들어 '네이버'라는 글자에 네이버 링크를 걸어보려 한다면 <.a href="https://www.naver.com/">네이버<./a> 형식으로 작성하면 된다.
직접 만들어보면 네이버 이렇게 된다.
새 탭에서 열기를 바란다면 시작 태그에 target="blank"를 작성하면 되고, 툴팁을 띄우기 위해서는 title="(툴팁내용)"을 작성하면 된다.

영상 연결하기

영상을 연결할 때에는 영상의 설명 부분에 있는 '공유' 아이콘을 선택한 후 '퍼가기'를 선택해서 복붙한다.

댓글 구현하기

데이터 베이스까지 구현하고 spam을 차단 기능이 필요하기 때문에 back end 기술이 필요하다. 너무 먼 길이니까 Disqus나 LiveRe를 사용하여 댓글 기능을 가져오도록 한다.
Disqus에서 가져오는 방법은 다음과 같다.

  1. 로그인한 후 관리자에서 Your Sites 클릭하고 New.
  2. 웹사이트 이름 작성 후 카테고리는 자유.
  3. Let's get start it! 클릭
  4. 아이콘 스루하고 맨 아래 Universal이라고 되어있는 링크 선택.
  5. 맨 위의 코드 복사 후 댓글 넣고 싶은 부분에 붙여넣기.
* 단, 이 기능은 웹 서버로 호스팅한 경우에만 사용 가능하다.

개발자와의 채팅 서비스 구현

tawk를 사용해서 구현 가능하고 방법은 다음과 같다.

  1. 사이트 접속 후 로그인을 한다.
  2. 왼쪽 아이콘 모음 중 가장 아래에 있는 톱니바퀴 아이콘을 클릭한다.
  3. Asset Default에서 Widget Code를 복사한 후 넣고 싶은 부분에 붙여넣으면 된다.
* 단, 이 기능은 웹 서버로 호스팅한 경우에만 사용 가능하다.

웹페이지 분석기 활용하기

Google Analytics 서비스를 활용해서 구현 가능하고 절차는 다음과 같다.

  1. 로그인한 후 New Account로 들어간다.
  2. Account Name은 개발자의 이름을 적는 부분이다.
  3. Website Name은 추적할 사이트의 이름을 말한다.
  4. Website URL은 추적할 사이트의 주소를 말한다. 깃허브로 연 주소를 넣으면 된다.
  5. Reporting Time Zone은 사용하는 현재 시각을 입력하는 것이다.
  6. Get Tracking ID 클릭 후 South Korea 선택 후 약관 동의 클릭.
  7. 이후 User Management에서 Tracking Code 선택, Global Site Tag 하단의 코드를 각 웹 페이지 Head 태그 안쪽에 붙여넣는다.

이후 audience 탭을 보면 languege, active users, location 등을 보고 방문자의 정보를 알 수 있다. technology 탭에서는 어떤 브라우저를 가장 많이 사용하는지 등 분석할 수 있는 데이터를 모을 수 있다.
